Submission

Homework solutions must be completed in LaTeX. You must submit a PDF version along with the LaTeX file and all supplemental files (code, images, etc) through Gradescope.

Each homework assignment has two corresponding Gradescope assignments - one for the writeup PDF and another for supplemental files.

When submitting the writeup PDF, you must assign pages for each question.
